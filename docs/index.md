# LegisPro

Bem-vindo à documentação do projeto LegisPro.

## Sobre o Projeto

O LegisPro é um sistema de gestão de demandas desenvolvido para escritórios de advocacia que buscam otimizar seu fluxo de trabalho. O projeto visa substituir o controle manual por planilhas por uma solução centralizada e visual, permitindo:

- Gestão Colaborativa: Distribuição clara de tarefas entre sócios e estagiários.
- Controle de Prazos: Acompanhamento rigoroso de datas críticas.
- Visibilidade Total: Uso de quadros Kanban para visualizar o status de cada demanda (Elaboração, Revisão, Concluído).

Este projeto está sendo desenvolvido como parte da disciplina de Requisitos de Software na Universidade de Brasília (UnB).

## O MVP (Produto Mínimo Viável)

O MVP do LegisPro foca em validar a gestão básica de demandas e a colaboração interna.

- Objetivo: Reduzir em até 50% o tempo gasto na coordenação de atividades.
- Funcionalidades Chave: Cadastro de demandas, Quadro Kanban configurável, Atribuição de responsabilidades e Controle de acesso por perfil (Sócio/Estagiário).
